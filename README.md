
# 🌐 Website Traffic Analysis - Alfido Tech

## 📌 Project Overview

This project focuses on analyzing website traffic logs to understand user behavior, user journeys, and engagement patterns. The goal is to identify key insights such as top landing pages, bounce rates, and traffic sources to improve website conversions.

---

## 🎯 Objectives

* Analyze user journeys across the website
* Identify top landing and exit pages
* Calculate key metrics like sessions, users, and bounce rate
* Understand traffic sources (Google, Direct, Social Media)
* Provide actionable recommendations to improve conversions

---

## 📊 Dataset Description

The dataset contains website traffic logs with the following columns:

* **user_id** – Unique user identifier
* **session_id** – Session identifier
* **page** – Page visited by the user
* **timestamp** – Date and time of visit
* **referrer** – Traffic source (Google, Direct, Facebook, etc.)

---

## 📈 Key Metrics Calculated

* Total Users
* Total Sessions
* Bounce Rate
* Average Session Duration
* Top Landing Pages
* Top Exit Pages
* Traffic Sources

---

## 📊 Visualizations

* Bar chart of top landing pages
* Pie chart of traffic sources
* User flow analysis

---

## 🔍 Key Insights

* Most users enter through the **Home page**
* Google is the primary traffic source
* High bounce rate observed in some sessions
* Blog pages attract users but have quick exits
* Product pages show better engagement

---

## 🚀 Recommendations

1. Improve homepage design with clear Call-To-Action (CTA)
2. Reduce bounce rate by adding internal links
3. Enhance product pages with detailed content
4. Use blog pages for lead generation
5. Optimize SEO and page loading speed
